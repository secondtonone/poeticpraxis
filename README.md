# POETIC PRAXIS App Frontend

Install mkcert (https://mkcert.dev/).

## Configure

```bash
npm install
npm run cert
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

### Local build

```bash
npm run build:local
```

### Local server run

```bash
npm run server:run
```

## Analyze bundle

```bash
npm run build:analyze
```
